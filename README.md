# Anima 🚀

An intelligent, empathic AI companion and development agent built natively for Obsidian. **Anima** breaks the static nature of traditional note-taking by introducing a reactive digital entity that lives in your sidebar, senses your emotional workflow, and adapts to your digital life operating system.

---

## ✨ Features (Current MVP)

* **Real-Time Emotional Reactivity:** Continuously listens to your active note modifications and changes its ASCII expressions dynamically based on your emotional state (e.g., shifts to `(╥﹏╥)` during stressful/sad moments or jumps to `(≧◡≦)` when positive milestones are written).
* **Multi-Provider AI Architecture:** Fully decoupled API integration, allowing users to safely bring their own API keys from any major provider (Google Gemini, OpenAI, etc.) directly within the built-in settings panel.
* **Minimalist Sidebar UI:** A distraction-free, beautiful dark-mode custom view built to integrate smoothly into your existing workspace layouts and terminal-inspired aesthetics.

---

## 🛠️ Tech Stack & Architecture

* **Language:** TypeScript / JavaScript (ES6+)
* **Environment:** Node.js / Electron (Obsidian API Wrapper)
* **Bundler:** Esbuild (Ultra-fast incremental development builds)
* **AI Integration:** Google Generative AI SDK & extensible multi-provider endpoints.
